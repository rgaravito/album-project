# Task: Database Schema & Data Models

## Description

Define and implement the core data models for cards, world cups, and user inventory using Prisma. Set up database migrations.

## Acceptance Criteria

- [ ] Card model with number, year, team, player, rarity
- [ ] WorldCup model for tournament information
- [ ] UserCard model for inventory tracking
- [ ] Transaction model for buy/sell/exchange tracking
- [ ] Prisma migrations created
- [ ] Relationships properly defined
- [ ] Seed script for development data
- [ ] Database initialized and tested

## Implementation Details

### Prisma Schema - `prisma/schema.prisma`

```prisma
model WorldCup {
  id        Int     @id @default(autoincrement())
  year      Int     @unique
  country   String
  totalCards Int
  createdAt DateTime @default(now())
}

model Card {
  id        Int     @id @default(autoincrement())
  number    Int
  year      Int     // World Cup year
  team      String
  country   String
  playerName String
  rarity    String  // \"common\" | \"rare\" | \"special\"
  createdAt DateTime @default(now())

  @@unique([year, number]) // No duplicates per year
}

model UserCard {
  id        Int     @id @default(autoincrement())
  userId    String
  cardId    Int
  quantity  Int     @default(1)
  
  status    String  @default(\"available\") // \"available\" | \"sold\" | \"exchanged\"
  listedForSale Boolean @default(false)
  price     Float?
  
  createdAt DateTime @default(now())
  updatedAt DateTime @updatedAt
  
  user      User    @relation(fields: [userId], references: [id], onDelete: Cascade)
  card      Card    @relation(fields: [cardId], references: [id])
  
  @@unique([userId, cardId])
}

model Transaction {
  id        Int     @id @default(autoincrement())
  type      String  // \"buy\" | \"sell\" | \"exchange\"
  
  fromUserId String
  toUserId   String?
  
  cardIds   Int[]   // JSON array of card IDs
  price     Float?  // For buy/sell
  
  status    String  @default(\"pending\") // \"pending\" | \"completed\" | \"cancelled\"
  
  createdAt DateTime @default(now())
  updatedAt DateTime @updatedAt
  
  fromUser  User    @relation(\"from\", fields: [fromUserId], references: [id], onDelete: Cascade)
  toUser    User?   @relation(\"to\", fields: [toUserId], references: [id], onDelete: SetNull)
}
```
