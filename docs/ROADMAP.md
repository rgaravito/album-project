# Product Roadmap - LáminaGol

## Phase 1: MVP (4-6 weeks)

### 1.1 Core Infrastructure
- [ ] Proyecto Node.js/Express básico
- [ ] Base de datos PostgreSQL con Prisma
- [ ] Autenticación JWT
- [ ] Frontend Next.js inicial

### 1.2 User Management
- [ ] Registro de usuarios
- [ ] Login/Logout
- [ ] Perfil de usuario
- [ ] Avatar básico

### 1.3 Sticker Catalog
- [ ] Modelo de datos para láminas adhesivas del álbum (Cards)
- [ ] Datos de Mundiales (years, countries)
- [ ] API de láminas por año
- [ ] Listado en frontend

### 1.4 Basic Marketplace
- [ ] Listar láminas para venta
- [ ] Compra simple
- [ ] Mi inventario
- [ ] Historial de transacciones

### 1.5 UI Básica
- [ ] Layout principal
- [ ] Página de explorador de láminas
- [ ] Marketplace view
- [ ] Mi colección

## Phase 2: Exchange & Enhancement (3-4 weeks)

### 2.1 Exchange System
- [ ] Sistema de intercambio
- [ ] Ofertas y contraoffertas
- [ ] Notificaciones básicas

### 2.2 Search & Filtering
- [ ] Búsqueda por nombre de jugador
- [ ] Filtrar por año/mundial
- [ ] Filtrar por país
- [ ] Ordenamiento (precio, rareza)

### 2.3 Advanced Inventory
- [ ] Estados de láminas (disponible, reservada, vendida)
- [ ] Duplicados automáticos
- [ ] Wishlist

### 2.4 Enhanced UI
- [ ] Diseño de láminas mejorado
- [ ] Modales de compra/venta
- [ ] Dashboard mejorado
- [ ] Mobile responsive

## Phase 3: Future (Post-MVP)

- [ ] Caché Redis para láminas
- [ ] Reseñas y ratings de usuarios
- [ ] Sistema de puntos/badges
- [ ] Chat en tiempo real
- [ ] Notificaciones por email
- [ ] Analytics
- [ ] Integraciones de pago avanzadas
- [ ] Mobile app

## Out of Scope (MVP)

- Sistema de puntos o gamification
- Reseñas y ratings
- Chat en tiempo real
- Pago real (simulado en MVP)
- Notificaciones por email
- App móvil nativa
- Machine learning para recomendaciones
